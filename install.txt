transact-qda — Installation


Prerequisites
-------------

  Docker Engine 24+ or Docker Desktop
  Node.js 20+ and npm
  Git
  bash (native on Linux/macOS; on Windows via WSL2)


Linux
-----

  1. Install Docker per https://docs.docker.com/engine/install/
     Add your user to the docker group (the shell replaces $USER with
     your account name automatically):

         sudo usermod -aG docker $USER
         newgrp docker

  2. Install Node.js 20 and Git.

     Debian/Ubuntu:
         curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
         sudo apt-get install -y nodejs git

     Fedora:
         sudo dnf install nodejs npm git

  3. Clone and configure:
         git clone https://github.com/joeriben/transact-qda.git
         cd transact-qda
         cp .env.example .env

     Set SESSION_SECRET in .env to a random string of 30+ characters.

  4. Install and start:
         npm install
         ./scripts/2_start_dev.sh


macOS
-----

  1. Install Docker Desktop from
     https://www.docker.com/products/docker-desktop/
     Launch it once and wait for startup to complete.

  2. Install Node.js and Git via Homebrew:
         brew install node git

  3. Clone and configure:
         git clone https://github.com/joeriben/transact-qda.git
         cd transact-qda
         cp .env.example .env

     Set SESSION_SECRET in .env to a random string of 30+ characters.

  4. Install and start:
         npm install
         ./scripts/2_start_dev.sh


Windows
-------

Run setup under WSL2.

  1. In PowerShell as administrator:
         wsl --install
     Restart when prompted. Set a username and password at first boot.

  2. Install Docker Desktop for Windows with the WSL 2 backend enabled.

  3. Open the Ubuntu terminal from the Start menu. Continue from step 2
     of the Linux section.


First run
---------

Open http://localhost:5174 in a browser.
Log in with username admin and password adminadmin.
Change the password at the yellow banner.

The first AI request downloads the embedding model
nomic-ai/nomic-embed-text-v1.5 (~150 MB) to .model-cache/.


Start and stop
--------------

Start:
    ./scripts/2_start_dev.sh

Stop:
    Ctrl+C in the terminal, then ./scripts/1_stop_all.sh


Updating
--------

    ./scripts/1_stop_all.sh
    git pull
    npm install
    ./scripts/2_start_dev.sh
